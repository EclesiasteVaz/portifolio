Resumo da minha carreira até 2024

Comecei em programação com os meus 15 anos, 2021, desde aí tenho adquirido imenso conhecimento. A principio fui programador desktop usando C#, criei sistemas de gestão para entrega de projectos escolares e pessoais, após de estudar colocar em prática os conhecimentos em C# parti para uma nova aventura que foi o desenvolvimento web, criei vários websites, landipages sistema de gestão(Digifarma), projectos com amigos e pessoais, não parei apenas no front-end mergulhei para o back-end e gostei. Trabalhei e trabalho com Laravel, PHP, MySQL, JavaScript, HTML5, CSS3. Mas após de todas essas areas e linguagens tenho uma paixão, se for assim que posso chamar por criar aplicativos, no fim de 2023 entrei no mundo de desenvolvimento mobile, com o Flutter. Onde magia acontece criando API com laravel e integrando com App Flutter que eu mesmo crio os dois.

Desenvolvimento Mobile

Trabalho com Flutter, dando a possiblidade de construir apps cross plataforma ou seja para Android e Ios, tem sido uma demanda de conhecimento incrível e até hoje continuo estudantando o flutter, criei um jogo Super Teclas, não esta na play store falta de meios e ainda poderá ter atualizações o jogo. E tenho trabalho em projecto anonimo, mas é incrível essa aventura e a minha area de foco.

Desenvolvimento Web Back-end

Durante um bom tempo da minha aventura em desenvolvimento web back-end, trabalhei com o PHP puro, para o projecto Digifarma, sistema de gestão para farmacias sendo um projeto autonomo, e trabalhei para outros projectos com a mesma tech, mas hoje já não continuo apenas com o PHP puro parti para um desafio maior.

Desenvolvimento Web Front-end

Ainda este ano criei um projecto de redesign do site da africell, projeto autonomo, não gostei imenso do design do site oficial deles então criei o projeto. É area que atuo imensas vezes na minha carreira, pois as landipages dos meus projectos eu mesmo é quem crio elas, com os conhecimentos que possuo em HTML5, CSS3 e JS, criei landipages para uma instituição academica de inglês. Essa area não trabalho com frames como React, Angular e Vue pois as areas em que me foco para trabalho formal é desenvolvimento web back-end e mobile.

Conselhos para os Devs

Escolhe uma area especifica na programação, seja ela web front-end, back-end, ou mesmo web fullstack, há outras areas como mobile, nativo ou hibrido, há também areas como modelagem e outros.
